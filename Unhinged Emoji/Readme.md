# Unhinged Emoji

Humorous unhinged emojis made with Genmoji on iOS.

### [> Download Emoji Images Zip Here <](https://github.com/ThioJoe/Resources-From-Videos/raw/main/Unhinged%20Emoji/Emoji-Images.zip)

## Notes: 
- There's an element of randomness in generation. You probably won't get the exact same images when you do it, but you can probably get pretty close.
- Some of the prompts might not seem to make sense or match the image. This is because I was trying to generate something else, but it happened to spit out the image shown, and I liked it so I saved it.
- Any "Unknown" prompts mean the prompt was lost somehow
- Any "REDACTED" prompts means I had to use a big brain workaround to get around the usual filters for that particular image. And I'm not going to give it away 😌 (they'd just block it if I did anyway).
